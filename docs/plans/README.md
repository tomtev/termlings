# Plans

This directory holds planning docs for future Termlings architecture.

Plans are the strategic layer above tasks and workflows.

Current plan docs:

- [PLAN-MODEL.md](PLAN-MODEL.md) - what plans are, when to use them, and how they relate to tasks and workflows
- [APP-ACTIVITY.md](APP-ACTIVITY.md) - shared activity output system for core apps and future custom apps

These files are forward-looking architecture docs, not stable end-user contracts.
