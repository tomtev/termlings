# Startup Team Template

PM-led startup team for day-to-day shipping.

## Agents

- **PM** — Vision & prioritization, day-to-day lead
- **Designer** — UX & visual design
- **Developer** — Build & ship
- **Growth** — Customer & adoption
- **Support** — Operations

## Usage

```bash
termlings init --template startup-team
```

## Customization

Edit agent SOUL.md files under `.termlings/agents/<slug>/SOUL.md`.
Update `.termlings/VISION.md` with your product direction.
