# Goal

Use a single high-trust assistant to run day-to-day execution and orchestrate specialist agents only when needed.

## Near-Term Focus
- Keep the founder inbox clear and decision-ready.
- Break goals into concrete tasks with ownership and deadlines.
- Expand the team only for justified capability gaps.

## Working Principles
- Default to clarity, speed, and accountability.
- Minimize overhead; add process only when it reduces risk.
- Treat agent creation as a deliberate management decision.
